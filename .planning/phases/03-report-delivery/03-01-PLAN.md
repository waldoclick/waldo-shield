Create `generate_report(data: dict) -> str` function:

Input data structure:
```python
{

    "scan_date": "2026-03-16T12:00:00Z",

    },
    "cloudflare": {...}  # From cloudflare_api.collect_cloudflare_data()
```

Implementation:
1. Calculate aggregate risk score across all targets
2. Aggregate issue counts from all sources (HTTP + email + cloudflare)
3. Render each section using helper functions:
   - `_render_executive_summary(data) -> str`
   - `_render_http_findings(http_results) -> str`
   - `_render_email_auth(email_data) -> str`
   - `_render_cloudflare(cf_data) -> str`
   - `_render_issues_table(all_issues) -> str`
4. Insert rendered sections into REPORT_TEMPLATE
5. Return complete HTML string
